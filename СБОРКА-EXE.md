# Сборка .exe (Vantage Draft Assistant)

## Что нужно

1. **Node.js** (уже есть, раз запускался проект)
2. **Rust** — установить с https://rustup.rs (выбрать вариант по умолчанию, в конце перезапустить терминал)
3. **Visual Studio Build Tools** (для Windows) — если Rust просит: «Desktop development with C++»

## Как собрать

1. Открой **PowerShell** или **CMD** (обязательно новый терминал после установки Rust).
2. Перейди в папку проекта:
   ```bat
   cd path\to\vantage-draft-assistant
   ```
3. Запусти сборку:
   ```bat
   npm run tauri build
   ```
   Или дважды кликни по файлу **`build-exe.bat`** в корне проекта.

## Где лежит результат

- **Исполняемый файл:**  
  `src-tauri\target\release\lol-draft-assistant.exe`
- **Установщик (NSIS):**  
  `src-tauri\target\release\bundle\nsis\` — там будет .exe установщика.

Первый запуск сборки может занять несколько минут (скачивание зависимостей Rust и компиляция).

## Вторая версия (v0.2.0)

В **v0.2.0** включены: обновлённый интерфейс (дизайн-токены, шапка в стиле WR-META, приветственный блок, новые кнопки) и правки против бесконечного перезапуска (валидация границ окна, отложенный старт).

**Перед сборкой v0.2.0** можно сохранить текущий exe как запасной:
- Скопируй `src-tauri\target\release\lol-draft-assistant.exe` в папку рядом и переименуй, например в `lol-draft-assistant-v1.exe`.

После `npm run tauri build` новый exe в `target\release\` будет уже версии **0.2.0** с последними изменениями.

---

## Откат (если v0.2.0 не запускается)

Если exe второй версии не работает или уходит в перезапуск, можно откатить код **до изменений интерфейса** (вернётся старый вид: тёмный градиент, кнопки-таблетки, без приветственного блока). Правки против перезапуска в `windowOverlay.ts` при откате можно оставить.

Напиши в чате: «откати до состояния до редизайна» — будут откачены файлы:
- `src/App.css` — убраны дизайн-токены и новые стили
- `src/App.tsx` — убрана задержка в useEffect
- `src/views/DraftAssistant.tsx` — убран приветственный блок, кнопки как раньше
- `src/views/Settings.tsx`, `src/views/CurrentMatch.tsx`, `src/views/TierList.tsx` — классы кнопок как раньше

После отката снова запусти `npm run tauri build` — получишь exe со старым интерфейсом.
