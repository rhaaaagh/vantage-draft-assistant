## v0.2.0 — 2026-06-25

Первый публичный релиз.

### Возможности
- **Драфт / Матч**: живой драфт из LCU (пики/баны, авто-роль), рекомендации пиков и
  сборки от собственного движка; составы и ранги текущей игры (Spectator); ручной
  подбор и анализатор чемпион-селекта.
- **Скаут**: разведка активной игры — составы, ранги, авто-роли, история, мастерство,
  личный и мета-матчап; «паттерны игрока» — карта смертей с таймингами, маршрут
  перемещений и лесные маршруты по матчам (линиями, со стрелками, по каждой игре).
- **Профиль**: ранг, винрейт, чемпионы, история, мастерство.
- **Тир-лист**: чемпионы по ролям и патчу; клик по имени открывает страницу чемпиона.
- **Чемпион**: винрейт/пикрейт/банрейт, матчапы, синергии, руны, путь сборки.
- **Краулер**: сбор статистики матчей (Challenger→Diamond) в локальную БД; держит
  текущий и предыдущий патч.
- **Разбор матча**: счёт, руны, графики золото/опыт/CS, карта смертей, объективы.

### Технически
- Tauri 2 (Rust) + React + TypeScript; локальная БД SQLite.
- Лимитер запросов Riot API с приоритетами; авто-определение региона из LCU.
- Актуальный миникап Summoner's Rift (CommunityDragon) с откатом на Data Dragon.
- Юнит-тесты на ядро (лимитер, парсинг, агрегаты, паттерны).
